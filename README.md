# ☕ Starbucks Beverage Nutrition Dashboard

## 📌 Project Overview

Built an interactive Power BI dashboard to analyze Starbucks beverage nutritional information and compare calories, caffeine, sugar, and beverage categories.

---

## 🎯 Business Problem

Consumers and businesses need nutritional insights to make healthier beverage choices.

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Excel

---

## 📊 Dashboard Features

- Total Beverages
- Calories Analysis
- Caffeine Analysis
- Sugar Analysis
- Beverage Categories
- Interactive Filters

---

## 📈 Key Insights

- Coffee beverages contain the highest caffeine.
- Smoothies generally contain higher sugar.
- Beverage preparation impacts nutritional values.
